# gphotofb
Linux Framebuffer display of random google photo picture

## Usage
```
$ gphotofb --help
Usage of ./gphotofb:
  -c string
        Credentials path (default "./credentials.json")
  -d string
        Path to framebuffer (default "/dev/fb0")
  -t string
        Rotation timeout (default "10s")
```

## Build
```
go build
```
